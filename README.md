### 사용법(Usage) 

한국어로:

1. `config.sample.py` 파일을 열어, 설정을 하세요.
2. filename 을 `config.py` 로 저장합니다.
3. `worker.py` 를 실행합니다.


in English:

1. Open the `config.sample.py`, then set your configuration.
2. Save this as `config.py`.
3. Run `worker.py`
